# 臺北版與 Sutton OpenBook 差距檢討

檢視日期為 2026 年 7 月 18 日，對照網站為 [Sutton OpenBook](https://finance.suttonma.gov/openbook) 與 [臺北市 OpenBook](https://mashbean.github.io/Open-Book/)。

## 第二版已改善項目

| 面向 | Sutton 原站 | 臺北第二版 |
| --- | --- | --- |
| 首頁敘事 | 市政透明主視覺、年度摘要、預算流程、支出排名、收支、資本計畫、文件入口 | 已採用相同資訊節奏與主要區塊 |
| 語言 | 英文 | 新增完整中英文切換，網址可用 `?lang=en` 分享 |
| 貨幣 | 美元 | 中文保留新臺幣，英文按固定匯率顯示美元 |
| 匯率透明度 | 原始資料即為美元 | 英文版清楚標示 `NT$32.25 = US$1`、日期、來源與估算性質 |
| 明細查詢 | 獨立支出、收入、資本等頁面 | 保留單頁快速切換，支援搜尋、年度比較與 CSV |
| 文件信任資訊 | 預算書、財務文件、免登入、定期更新等入口 | 新增資料與文件入口及四項信任資訊 |
| 國際讀者資料 | 英文名稱及美元 | 主管機關、歲入分類、科目、來源、FAQ 與 CSV 均已英文化 |

## 仍存在的主要差距

| 差距 | 原因 | 後續需要 |
| --- | --- | --- |
| 缺少市政實景照片與正式市徽 | 目前未納入可確認授權的圖像資產 | 取得臺北市政府可再利用的官方照片與識別規範 |
| 缺少準備金頁 | 現有資料只有第二預備金預算，無法代表完整財政穩定基金 | 取得準備金、基金餘額與決算資料 |
| 資本門只到主管機關 | 官方機器可讀總預算沒有個別工程名稱、位置與期程 | 串接資本計畫書或公共工程資料，建立專案識別碼 |
| 歲出缺少功能別與逐項明細 | 現有首頁採主管機關彙總，粒度低於 Sutton 的功能別與 line items | 納入政事別資料及歲出計畫、用途別明細 |
| 缺少獨立預算書閱讀器 | GitHub Pages 版本尚未處理大型 PDF 與章節索引 | 建立預算書文件清單、章節連結與可搜尋文字層 |
| 導覽仍是單頁切換 | 靜態版以 hash 保存頁面狀態 | 若需要完整搜尋引擎索引，可改為多頁靜態輸出 |
| 收支差額缺少法定平衡說明 | 歲入與歲出資料集的口徑需進一步核對融資與移用項目 | 對照總預算案總說明與融資調度欄位 |

## 美元換算規則

英文版把官方新臺幣金額除以 `32.25`。此數值為臺灣銀行 2026 年 7 月 17 日美元即期買入與賣出牌告匯率中價，僅供國際比較。CSV 同時保留美元估算、新臺幣原值與換算率。
